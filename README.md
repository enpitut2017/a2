## A1 「つくByeBuy(仮)」
筑波大生の品物手渡しマッチングサービス
## エレベーターピッチ
「教科書や家具を買うのに手間とお金がかかる問題」を解決したい「家の近くに住んでいる人から家具等が欲しい筑波大生」向けの「スムーズな取引」を実現する「つくByeBuy(仮)」です。これは「欲しいものを検索して取引する」ことができ、「ジモティー、ツクナビの掲示板など」と違って「天3、春4などのピンポイントな地域で検索できる機能や、学類で教科書を検索する機能」が備わっています。
